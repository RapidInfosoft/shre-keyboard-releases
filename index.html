Install

  1. Tap Download APK above (in Chrome on your phone).
  2. When prompted, allow your browser to install unknown apps, then tap Install.
  3. Open the app once — it shows the setup screen.

Enable the keyboard

  1. Go to Settings ▸ System ▸ Languages & input ▸ On-screen keyboard ▸ Manage keyboards.
  2. Turn on Shrey Keyboard.
  3. In any app, tap a text field, then tap the 🌐 / keyboard icon to switch to Shrey.
  4. Top bar: ⌨ Keys · 🎤 Voice · Aa Tone. Tap Aa to cycle the writing tone (Formal → Professional → Friendly → Casual → …).

Tip: open the app's Settings screen to set your AI URL/key, pick a default tone, and manage your custom dictionary.
