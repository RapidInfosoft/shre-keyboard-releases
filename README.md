## Install

1. Download the APK from the link above **on your Android phone**
2. Allow your browser to *install unknown apps* when prompted (Android's standard check for apps outside Google Play), then tap **Install**
3. Enable the keyboard — the path differs by device:
   - **Stock Android (Pixel & most others):** Settings › System › Languages & input › On-screen keyboard › Manage on-screen keyboards → turn on **Shre Keyboard**
   - **Samsung Galaxy:** Settings › General management › Keyboard list and default → turn on **Shre Keyboard**
4. Tap any text field, open the keyboard switcher (globe key), and pick **Shre Keyboard**

Full walkthrough: [landing page → Install](https://rapidinfosoft.github.io/shre-keyboard-releases/#install)
