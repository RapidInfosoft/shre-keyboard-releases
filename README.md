# Shre Keyboard — Beta Downloads

AI-powered voice keyboard for Android with Shre AI platform integration.

## Download

**[Download Latest APK](https://github.com/RapidInfosoft/shre-keyboard-releases/releases/tag/latest)**

## Features

- Voice-to-text with AI cleanup (50+ languages)
- Smart Compose — context-aware text generation
- Ask Shre — talk to any Shre agent from any app
- Auto-translate: speak in any language, insert in English
- 3-tier voice engine: On-device → NVIDIA Riva (GPU) → Whisper (cloud)

## Install

1. Download `shre-keyboard-latest.apk` from the link above
2. Tap to install (enable "Install from unknown sources" if prompted)
3. Go to **Settings > System > Languages & Input > On-screen keyboard**
4. Enable **Shre Keyboard**
5. Open the Shre Keyboard app, enter your API URL + Key
6. Switch to any app and use the keyboard switcher to select Shre Keyboard
